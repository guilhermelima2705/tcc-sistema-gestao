# TCC - Sistema de Gestão Salão de beleza 🚀

## 📝 Sobre o Projeto
Este é um sistema de gestão desenvolvido como Trabalho de Conclusão de Curso (TCC). O objetivo é oferecer uma solução robusta para automação de processos, agendamentos e controle financeiro para estabelecimentos de estética e beleza. O projeto foca em escalabilidade, segurança e uma arquitetura limpa.

---

## 👨‍💻 Meu Papel (Liderança Técnica)
Como líder do projeto, sou responsável por:
- **Arquitetura Backend:** Estruturação do projeto utilizando Java 21 e Spring Boot.
- **Modelagem de Dados:** Design do banco de dados relacional com PostgreSQL.
- **Segurança:** Implementação de autenticação e autorização com Spring Security e JWT.
- **Infraestrutura:** Containerização da aplicação utilizando Docker para facilitar o deploy e ambiente de desenvolvimento.
- **Gestão de Time:** Coordenação das tarefas e revisão de código (Code Review).

---

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Java 21
- **Framework:** Spring Boot 3+ (JPA, Security, Validation)
- **Banco de Dados:** PostgreSQL
- **Containerização:** Docker & Docker Compose
- **Gerenciador de Dependências:** Maven
- **Versionamento:** Git

---

## 🏗️ Estrutura do Projeto
O projeto segue o padrão de camadas para melhor manutenção:
- `Controller`: Endpoints da API REST.
- `Service`: Regras de negócio.
- `Repository`: Comunicação com o banco de dados.
- `DTO`: Objetos de transferência de dados para segurança e performance.
